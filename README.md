# OMS
小微企业跨组织人才管理系统
***
## Introduction
OMS:Inter Organizational talent Management System for small and micro enterprises.

This project is developed by Thinker group, used for joining the competition named *China Students Service Outsourcing Innovation and Entrepreneurship Competition, 2021*. It aimed to create a Management System to provide Convenient Service between HR and Staff.

## Framework
We follow the idea of separation of front-end and back-end to develop this project.

The Back-end program is in folder *API*. They should be deployed on a PHP server. And the Front-end is in folder *web*. They're written in html+css+js and should be deployed on a static web server.
